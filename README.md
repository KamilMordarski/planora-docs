# Dokumentacja Planory

Oficjalne repozytorium strony dokumentacji aplikacji
[Planora](https://github.com/KamilMordarski/planora).

- [Otwórz dokumentację](https://kamilmordarski.github.io/planora-docs/)
- [Pobierz najnowszą Planorę dla Windows](https://github.com/KamilMordarski/planora/releases/latest/download/Planora-latest.exe)
- [Zobacz wszystkie pliki wydania](https://github.com/KamilMordarski/planora/releases/latest)

Strona jest statyczna i modularna:

- `index.html` — treść dokumentacji,
- `assets/styles.css` — wygląd i responsywność,
- `assets/app.js` — wyszukiwarka i nawigacja,
- `assets/planora-icon.png` — ikona aplikacji.

Publikacja na GitHub Pages odbywa się automatycznie po zmianach w gałęzi `main`.
